# Profinance

## Description

Profinance app

## Usage

Install all required packages:

```bash
npm install
```

Run the application:

```bash
npm run dev
```

Open http://localhost:5173 to view it in the browser.

## Scripts

Build the application:

```bash
npm run build
```

Launch the unit-test runner:

```bash
npm run test:unit
```

Launch the e2e-test runner:

```bash
npm run test:e2e
```

Launch the linter:

```bash
npm run lint
```
